# BrainTwist
An implementation of the BrainF**k language.

## Hello World Usage Example
```
$code = '>+++++++++[<++++++++>-]<.>+++++++[<++++>-]<+.+++++++..+++.[-]>++++++++[<++++>-]
         <.#>+++++++++++[<+++++>-]<.>++++++++[<+++>-]<.+++.------.--------.[-]>++++++++[
         <++++>-]<+.[-]++++++++++.';
$bt = new BrainTwist();
$result = $bt->interpret($code, '');
echo $result;
```
